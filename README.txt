Herramienta de homologación y fusión de mezclas

Contenido:
- index.html -> interfaz principal
- app.js -> lógica de homologación, filtros, cargas nuevas y pronóstico
- styles.css -> estilos
- initial-data.js -> base inicial homologada con archivos enero, febrero y marzo 2026
- base_homologada_inicial_mezclas.xlsx -> export inicial en Excel

Notas:
1. La herramienta ya carga la base homologada inicial.
2. Las nuevas cargas se conservan en el navegador mediante localStorage.
3. El cálculo exacto de piezas depende del archivo de piezas por empaque.
4. La lógica sigue la guía cargada en esta conversación.
